This project is based on the project in https://www.geeksforgeeks.org/machine-learning/stock-price-prediction-using-machine-learning-in-python/

The goal of this project is to create a stock price predictor using past stock data

Note 1: The data that will be used is from TSLA market data over the past year: https://www.marketwatch.com/investing/stock/TSLA/download-data?startDate=1/20/2025&endDate=1/21/2026
